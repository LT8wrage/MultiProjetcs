# MultiProjetcs
- These projects have been coded with python, please proceed to download it ( https://www.python.org/downloads/ ) and download a convinient IDE  ( https://code.visualstudio.com/Download or https://www.jetbrains.com/pycharm/download/ )
- If you want to access the any of the projects, please intall the requirements via the following command: "pip install -r requirements.txt" then execute app.py

## Including:
### Number recognition AI from handwritten number using python:
- used MNIST datasets from: https://www.kaggle.com/datasets/hojjatk/mnist-dataset
- separated the project in two files: AI.py to train my ai using MNIST datasets and tensorflow then saved the ai model into a keras file
- GUI.py for the user to draw and sumbit hand-drawn numbers using tkinter (the ai is still a bit dumb, please forgive it)

### 3D objects rotation script:
- was originally a complex geometry dash project ( building a 3d graphical engine inside a 2d game using only 2 types of triggers, explanatory video on: https://www.youtube.com/channel/UCS8M2_Nsh23_3O7hmYzz5uA )
- can rotate the following objects : cube, octahedron, dodecahedron
